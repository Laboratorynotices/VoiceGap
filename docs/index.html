Инструкция по использованию приложения VoiceGap

Назначение приложения

VoiceGap - это инструмент для подготовки текстов к диктанту с помощью нейросети. Приложение автоматически добавляет паузы между словами в зависимости от их длины, что делает диктант более удобным для восприятия и записи.

Основные функции

Ввод текста

  1. Введите или вставьте исходный текст в левое поле ввода.
  2. Приложение автоматически обработает текст и добавит необходимые паузы.
  3. Результат появится в правом поле вывода.

Статистика

В разделе статистики вы увидите:

  • Общее количество слов в тексте
  • Общее количество символов
  • Рекомендуемое время для записи диктанта учениками

Настройки генерации пауз

Вы можете настроить параметры генерации пауз:

  • Минимальная длина паузы - минимальное количество миллисекунд паузы
  • Максимальная длина паузы - максимальное количество миллисекунд паузы
  • Минимальная длина слова - слова с меньшим количеством символов будут иметь минимальную паузу
  • Максимальная длина слова - слова с большим количеством символов будут иметь максимальную паузу

Порядок работы с приложением

  1. Откройте приложение VoiceGap.
  2. Настройте параметры генерации пауз в соответствии с вашими требованиями.
  3. Введите или вставьте текст в поле ввода.
  4. Проверьте результат в поле вывода - текст будет содержать специальные метки пауз.
  5. Скопируйте обработанный текст из поля вывода.
  6. Вставьте текст в нейросеть LuvVoice (https://luvvoice.com/) для озвучивания.
  7. Используйте полученный аудиофайл для проведения диктанта.

Советы по использованию

  • Перед обработкой большого текста рекомендуется протестировать настройки на небольшом фрагменте.
  • Для младших классов можно установить более длительные паузы.
  • При необходимости вы можете вручную отредактировать текст в поле вывода перед отправкой в нейросеть.
  • Используйте данные статистики для планирования времени урока.

Примечание

В будущих версиях планируется добавить возможность выбора нейросети для более точной настройки синтаксиса пауз.
